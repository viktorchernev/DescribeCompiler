DescribeParser.Visitors namespace - Log Visitor for Describe 1.1

1. LogVisitor11

This visitor class is used to log to the console the result of a parse operation. This LogVisitor is for Describe version 1.1

Log - Gets or sets all the logs for this instance.
LastError - Gets the last error for this instance. If there was an error, this string contains the message, and if there weren't any errors, this will be `null`.
